## A. Ask
- 절대로 정답을 만드는 작업이 아님.
- 어? 혹시 이럴까? 수준의 궁금함에서 시작해야함
1. 현재의 고객중 어떤 특성을 지닌 고객군이 매출에 상대적으로 큰 기여를 하고 있는가?
2. 고객 특성에 따라 구매 패턴이 어떤 차이를 보이는가?
3. 온라인과 오프라인 채널의 매출 규모 및 구조는 어떤 차이를 보이는가?
4. 상품군별(색상, 타입 등) 특성은 어떻게 구성되어 있고, 매출이 어떻게 분포되어 있는가?
